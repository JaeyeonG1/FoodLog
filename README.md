## FoodLog 소개
### 기획 의도
- 코로나19로 인한 불규칙한 생활습관
- 체계적인 식단 관리 중요성
- 간편하게 식단 관리
### 주요 기능
- 식단 이미지 촬영
  - 예측 결과 확인
  - 결과 수정 및 저장
  - 사용자 피드백

- 식단 기록
  - 음식 이미지 crop
  - 일별, 시간별 기록
